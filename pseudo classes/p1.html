CSS PSEUDO-CLASSSES....!

The css PSEUDO-CLASSSES allows u to style the dynamic states of an element such as hover, active and focus states, as well as elements that exisiting in the doc tree but can't be targeted via the use of onther selectors without adding any IDs or classes to them, for example, targeting the first or last child elements.

A PSEUDO-CLASSSES starts with a colon (:). Its syntax can be given with :

Selectors:psudo-class {Property : value}
