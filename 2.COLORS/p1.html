Defining color values

Colors in CSS most often specific in the following formate..

  1. Color Keyword - like red,green,blue,orange and etc..
  2. Hexadecimal (2^8 = 256) values - like #000000, #ffffff and etc...
  3. RGB (RED, GREEN, BLUE) Value - Like rgb(0,0,0),rgb(255,0,0)
  4. RGBA (RED, GREEN, BLUE, ALPHA) Value - Like rgb(0,0,0,0.5),rgb(255,0,0, 1)
  5. HSL (HUE, SATURATION, LIGHTNESS) values
  6. HSLA (HUE, SATURATION, LIGHTNESS, ALPHA) values
