CSS PSEUDO-ELEMENT

The CSS PSEUDO-ELEMENT allow u to style the element or part of the element without adding any id's or classes to them

CSS3 introduced a new double-colon (::) syntax for PSEUDO-ELEMENT to destinguish between them and PSEUDO-class

selector :: PSEUDO-ELEMENT {property : value}

::first-letter and ::first-line
